# BTLE-Remote-Car

Remote car driven by Xamarin App over Bluetooth Low Energy

Project elements:
* STM32F303RE dev nucleo board
* BlueNRG-M2SP bluetooth module (X-NUCLEO-BNRG2A1 shield for nucleo)
* Xamarin Mobile App
